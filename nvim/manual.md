# 🧭 Neovim キーマップマニュアル

## 🧩 基本情報

この設定では `Map()` 関数を使って共通のオプション（`noremap = true`）を指定しつつ
ノーマルモード・インサートモード・ビジュアルモードに対してカスタムマッピングを登録しています。

---

## 🎮 ノーマルモード (`n`)

| キー     | 動作内容                           | 備考                              |
| -------- | ---------------------------------- | --------------------------------- |
| `<C-s>`  | ファイルを保存                     | `:w` コマンド実行                 |
| `<C-d>`  | 現在の値をデクリメント             | 標準の `<C-x>` と同じ動作         |
| `<C-v>`  | システムクリップボードから貼り付け | `"*` レジスタ利用                 |
| `p`      | ペースト後に位置を調整             | `]p` によりカーソルの整列が自然に |
| `H`      | 行頭へ移動                         | `0`                               |
| `J`      | 10 行下へ移動                      | 連続移動用                        |
| `K`      | 10 行上へ移動                      | 同上                              |
| `L`      | 行末へ移動                         | `$`                               |
| `<CR>`   | 次の行を開く                       | `o<Esc>` と同等                   |
| `<S-CR>` | 前の行を開く                       | `O<Esc>` と同等                   |
| `<BS>`   | 1 文字削除モードに                 | `i<BS><Esc>`                      |
| `U`      | アンドゥをリドゥ                   | `<C-r>`                           |
| `==`     | 全体の整形 (`gg=G''`)              | `=` による自動インデント          |

## 📐 ビジュアルモード (`v`)

| キー                  | 動作内容                               | 備考                |
| --------------------- | -------------------------------------- | ------------------- |
| `v`                   | ビジュアルブロックモードへ             | `<C-v>`             |
| `<C-c>`               | 選択範囲をコピー                       | クリップボード `"*` |
| `<C-x>`               | 選択範囲を切り取り                     | `"*` レジスタ利用   |
| `,`                   | 全選択                                 | `ggVG`              |
| `jk`                  | ノーマルモードに戻る                   |                     |
| `H` / `J` / `K` / `L` | それぞれ行頭／10 行下／10 行上／行末へ |                     |

## 📖 備考

- `<leader>` はデフォルトでは `\` か `Space`（設定に依存）
- 主要なキーは「直感的」かつ「連続操作を効率化」するよう整理されています
- 特に `HJKL` の再割り当てで行頭・行末・10 行移動が統一されているのが特徴
